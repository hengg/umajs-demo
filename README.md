# Umajs-demo
Umajs 使用示例

[在线文档](https://umajs.gitee.io/)

[源码地址](https://github.com/wuba/Umajs)

## Usage

安装依赖：
```
npm i
```
或者
```
yarn
```
启动项目：
```
npm run dev
```

项目启动后，您可以通过访问以下 URL 体验 Umajs：
- [http://localhost:8058](http://localhost:8058)
- [http://localhost:8058/tpl/test](http://localhost:8058/tpl/test)
